🗺️ Manage journeys

A journey is the full settlement path for one visa type in one UK region. Create it here first — a journey needs at least one phase before tasks can be assigned to it.

Days are counted from the newcomer's arrival date, e.g. 0–7 for "First week".

Existing journeys

Create a task

Add a new settlement task and assign it to a phase and visa type.

Hold Ctrl (Cmd on Mac) to select more than one. This task stays hidden from newcomers until every selected task is marked complete.

New tasks start as drafts. Use "Submit for review" and "Publish" in the list below to move a task through the workflow, or set it directly here.

Existing tasks

Reorder tasks within a phase

This controls the order newcomers see tasks of the same urgency in a phase. Critical always shows before Important, and Important before Optional — reordering here only breaks ties within one urgency tier, so it can't bury a Critical task under Optional ones.
